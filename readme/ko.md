## 시작하기

### 조건

- Node.js >= 18.0.0
- npm >= 9.2.0
- pnpm@7.26.3

> npm run init  
> npm run start:tavern

그리고

> npm run webpack  
> Chrome에서 불러오기 (dist 폴더를 선택하세요.)
